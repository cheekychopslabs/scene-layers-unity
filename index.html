Scene Layers for Unity

Photoshop-style layer management for Unity scenes

Get it on Unity Asset Store

Features

  • Organize scene objects into layers independent of hierarchy
  • Toggle visibility and picking for entire layers
  • Auto-assign rules based on component types
  • Save and load layer views (presets)
  • Color-coded hierarchy highlighting
  • Drag-and-drop workflow
